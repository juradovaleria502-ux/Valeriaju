LA TEGNOLOGIA

La tecnología es la aplicación del conocimiento científico para crear herramientas, técnicas y métodos que resuelven problemas y satisfacen necesidades humanas, buscando mejorar la calidad de vida. Proviene de las palabras griegas téchnē (arte, oficio) y logía (estudio o tratado). No es solo un conjunto de objetos físicos (como los computadores), sino también un proceso que abarca desde las habilidades para crear objetos hasta los sistemas de software y las prácticas para optimizar procesos.
